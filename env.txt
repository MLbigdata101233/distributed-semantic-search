### 1. Download the Miniconda Installer
First, we will create a folder for it and download the latest installation script directly from the official repository.

```bash
mkdir -p ~/miniconda3
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh -O ~/miniconda3/miniconda.sh
```

### 2. Run the Installer
Next, execute the script. The flags used here (`-b`, `-u`, `-p`) tell it to run automatically without asking you to press "Enter" repeatedly, and to install it in the folder we just created.

```bash
bash ~/miniconda3/miniconda.sh -b -u -p ~/miniconda3
```

### 3. Clean up the Installer File
Once it finishes installing, you don't need the installation script taking up space anymore, so you can safely delete it.

```bash
rm -rf ~/miniconda3/miniconda.sh
```

### 4. Initialize Conda
This is a crucial step. It links the `conda` command to your terminal so you can use it from anywhere. 

```bash
~/miniconda3/bin/conda init bash
```
*(Note: If your server uses `zsh` instead of `bash`, change the end of that command to `zsh`).*

### 5. Apply the Changes
For the initialization to take effect, you need to reload your terminal profile. You can either log out of the server and log back in, or simply run this command:

```bash
source ~/.bashrc
```
You should now see `(base)` next to your terminal prompt, which means Conda is active!

### 6. Verify the Installation
To make absolutely sure everything worked, check the version:

```bash
conda --version
```
